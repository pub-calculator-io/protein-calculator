Relying on certain statistics and recommendations, this free protein calculator predicts how much protein a person needs each day to be healthy.

[https://www.calculator.io/protein-calculator/](https://www.calculator.io/protein-calculator/)

== Usage ==

Add the Protein Calculator shortcode to your page, post or sidebar:

`[ci_protein_calculator]`

Add the following code to your template where you would like the Protein Calculator to appear:

`<?php display_ci_protein_calculator(); ?>`

== Installation ==

1. Upload the Protein Calculator /ci_protein_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Protein Calculator plugin through the "Plugins" menu in WordPress.
